# AutoWorldGen

AutoWorldGen is a Blender script for procedural terrain generation and erosion simulation.

## Features

- Procedural terrain generation using Perlin noise.
- Erosion simulation for realistic landscapes.
- Customizable parameters for terrain and erosion.
- Visualization of the erosion process within Blender.

## Requirements

- Blender 2.8 or higher.
- Python 3.x.
- Required Python packages: `numpy`, `concurrent.futures`.

## Usage

1. Open Blender.
2. In the Text Editor, load `main.py`.
3. Run the script to generate terrain and simulate erosion.

## Files

- [main.py](./main.py): Main script for terrain generation and erosion simulation.
